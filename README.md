# CHAT_BOT_RAG

This is a simple chat bot using Qwen-2.5-Instruct (open source) model. It is a rag based chatbot where it takes two urls  as its input and stores
the content in the faiss vectorstore. Then chunk has been created from the data structure and feed it to the model (Qwen). And the model can answer
any of the question you asked from the URLs link. 
The bot is a little bit slow i.e. it has less parameters and it runs locally so these factors affect the working speed of chatbot.
But it is working fine....
